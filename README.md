# AI Marketing Automation Agent

An AI-powered automation system that generates and publishes marketing content across multiple platforms.

## 🚀 Features

- Automated blog content generation
- Keyword research automation
- LinkedIn post creation
- YouTube description generator
- Amazon Affiliate content integration
- GitHub Actions automation workflow
- Web-based dashboard control panel

## 📊 Platforms Supported

- Blogger
- LinkedIn
- YouTube
- Amazon Affiliate

## ⚙️ Project Structure

ai-marketing-agent/
│
├── dashboard/
│   └── index.html              # Web dashboard
│
├── .github/workflows/
│   └── automation.yml          # Automation workflow
│
├── content_generator.py        # AI content generator
├── keyword_finder.py           # Keyword research tool
├── linkedin_post.py            # LinkedIn content generator
├── youtube_description.py      # YouTube SEO description tool
├── publish_blogger.py          # Blogger auto-publisher
│
└── keywords.txt                # Target marketing keywords

## 🤖 Automation Workflow

The system automatically:

1. Finds trending keywords
2. Generates blog content
3. Creates LinkedIn posts
4. Generates YouTube descriptions
5. Adds affiliate marketing links
6. Publishes content automatically
